✅ Ce que Localify n’est pas

Ce n’est pas un service de streaming public. C’est un serveur personnel, pensé pour votre propre bibliothèque.

🎯 Usage perso 🏠 Réseau local 🌐 Accès privé

Démarrage rapide

1

Installez Node.js LTS puis vérifiez avec node -v et npm -v.

2

Dans le dossier du projet, lancez npm install.

3

Démarrez l’app avec npm start puis ouvrez http://localhost:3000.

4

Ajoutez vos fichiers dans music/Artiste/Album et connectez-vous (admin / admin123456 au 1er démarrage).

FAQ

Puis-je utiliser Localify depuis mon téléphone ?

Oui. Ouvrez l’URL du serveur dans votre navigateur mobile, puis ajoutez-la à l’écran d’accueil pour un usage PWA.

Comment accéder à distance en sécurité ?

Le plus simple est d’utiliser un réseau privé comme Tailscale plutôt qu’une exposition publique du port.

Que faire si la bibliothèque est vide ?

Vérifiez la structure music/Artiste/Album, les extensions audio supportées et lancez un rescan depuis Paramètres.

Pourquoi un diagnostic système ?

Pour voir rapidement l’état du serveur (uptime, cache, index musique) et faciliter le dépannage.
